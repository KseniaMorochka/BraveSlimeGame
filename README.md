### Сюжет:<br/>
Храброму слайму нужна помощь. Ему необходимо собрать пять алмазов, чтобы вернуться домой. Нужно помогать слайму, проходя уровни, в конце которых ждет ценная награда, которая поможет слайму вернуться домой.
